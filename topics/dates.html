Working with dates & times

Python's built-in datetime module handles dates, times, timezones and formatting.

Examples

from datetime import datetime, date, time, timedelta

now = datetime.now()
today = date.today()
d = date(2025, 12, 31)
tomorrow = today + timedelta(days=1)

# format
s = now.strftime("%Y-%m-%d %H:%M:%S")
# parse
dt = datetime.strptime("2025-12-31", "%Y-%m-%d")

Timezones

Use zoneinfo (Python 3.9+) or third-party pytz for timezone-aware datetimes.

Common tasks

  • Difference between datetimes: subtract to get timedelta.
  • Scheduling, formatting user-friendly dates, converting between zones.
