Matplotlib — plotting in Python

Matplotlib is the standard 2D plotting library. Use it for quick visualizations and static charts.

Simple plot

import matplotlib.pyplot as plt

x = [1,2,3,4]
y = [10, 20, 15, 30]
plt.plot(x, y, marker='o')
plt.title("Sample plot")
plt.xlabel("x")
plt.ylabel("y")
plt.savefig("plot.png")   # save to file
plt.show()

Scatter, bar, histogram
