pip — Python package installer

pip installs and manages packages from PyPI.

Common commands

pip install requests          # install package
pip install "package==1.2.3"    # install specific version
pip install -U package          # upgrade
pip uninstall package

Requirements file

# requirements.txt
requests==2.31.0
flask

# install
pip install -r requirements.txt

Best practices

  • Use a virtual environment per project.
  • Pin versions in production with a `requirements.txt` or `pip freeze > requirements.txt`.
  • Prefer `pip` from an activated venv to avoid global installs.
