Dictionaries

Dictionaries store data in key–value pairs. They are unordered, mutable and optimized for fast lookups.

Creating dictionaries

student = {
    "name": "Asha",
    "age": 20,
    "marks": 88
}

print(student["name"])   # Asha
print(student.get("age"))

Adding & updating

student["city"] = "Pune"
student["marks"] = 90   # update value

Looping through dicts

for key in student:
    print(key, student[key])

for key, value in student.items():
    print(key, value)

Exercises
