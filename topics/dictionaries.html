Dictionaries (dict)

Dictionaries store key → value mappings. Keys are typically strings or numbers and must be immutable. Values can be anything.

Create & access

d = {"name":"Ada", "age":30}
print(d["name"])       # Ada
print(d.get("email"))  # None (safe access)

Modify

d["age"] = 31
d["city"] = "Bengaluru"
del d["city"]

Iterate

for k in d:
    print(k, d[k])

# keys(), values(), items()
for k, v in d.items():
    print(k, v)

Common uses

  • Counting/frequency maps
  • Configuration objects
  • Storing structured data (records)
# count words
words = ["a","b","a"]
counts = {}
for w in words:
    counts[w] = counts.get(w, 0) + 1

Exercises

  1. Write a function that inverts a dict (values → keys). What happens if values are not unique?
  2. Given a list of people (name, score), build a dictionary mapping names to scores.
