Casting (Type Conversion)

Casting means converting one data type into another using built-in functions.

Common Conversions

x = int("10")      # str → int
y = float("3.14")  # str → float
s = str(99)        # int → str

Errors

int("hello")   # ValueError

Exercises

  1. Convert user input (string) into an integer.
  2. Try converting “5.8” into int. What happens?
