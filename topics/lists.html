Lists

Lists are ordered, mutable sequences. They are versatile and one of the most-used data structures in Python.

Details

Create lists with square brackets. Common methods: append, extend, insert, pop, remove, sort, reverse. List comprehensions provide concise syntax to transform lists.

Examples

fruits = ['apple','banana']
fruits.append('kiwi')
fruits.insert(1,'orange')
print([f.upper() for f in fruits])  # comprehension
fruits.sort()

List methods (quick)

MethodEffect
append(x)Add x to end
extend(iter)Extend by iterable
pop([i])Remove and return item
sort()In-place sort

Common mistakes

  • Mutating a list while iterating over it can skip elements.
  • Copying a list by assignment b = a creates an alias; use a.copy() or a[:] to clone.

Exercises

  1. Given a list, remove duplicates while preserving order.
  2. Use a comprehension to produce squares of numbers 1–10.
