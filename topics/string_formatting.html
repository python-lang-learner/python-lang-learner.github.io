String Formatting in Python

Python provides multiple ways to format strings when inserting variables, numbers, and expressions. The most common and recommended method is using f-strings.

1. F-Strings (recommended)

Introduced in Python 3.6, f-strings are fast, readable, and clean.

name = "Alice"
age = 25

print(f"My name is {name} and I am {age} years old.")
# My name is Alice and I am 25 years old.

Expressions inside f-strings

print(f"Next year I will be {age + 1}.")

Formatting numbers

pi = 3.141592

print(f"{pi:.2f}")     # 3.14
print(f"{10000:,}")    # 10,000
print(f"{0.345:.1%}")  # 34.5%

2. .format() method

print("Hello {}, you scored {} marks".format("Ravi", 95))

Number formatting

print("Value: {:.2f}".format(12.56789))

Positional arguments

print("{1} is older than {0}".format("Arun", "Meera"))

3. Percent formatting (older style)

name = "Tom"
marks = 88

print("Student: %s, Marks: %d" % (name, marks))

This method is old and not recommended for new Python code, but you may still see it in legacy projects.

4. Format Specifiers (all styles)

Common specifiers:

  • :.2f → 2 decimal places
  • :, → thousand separator
  • :05d → padded integer (e.g., 00042)
  • :.1% → percentage
  • :>10 → right aligned
  • :<10 → left aligned
  • :^10 → centered
print(f"{42:05d}")      # 00042
print(f"{1234:,}")      # 1,234
print(f"{0.567:.1%}")   # 56.7%
print(f"{'Hi':^10}")    # "    Hi    "

5. Multiline formatting

name = "Sana"
city = "Pune"

msg = f"""
Name: {name}
City: {city}
Welcome!
"""

print(msg)

Exercises

  1. Format a price value to always show exactly 2 decimal places.
  2. Create an f-string that prints name, age and a greeting message.
  3. Align 3 words in left, right and center positions using formatting.
