Data Types

Python has built-in scalar types (int, float, str, bool, None) and collection types (list, tuple, set, dict). Understanding mutability and operations is important.

Details

Scalars: int (integers), float (decimal numbers), str (text), bool (True/False), and None (no value). Collections: list (ordered, mutable), tuple (ordered, immutable), set (unordered, unique), dict (key→value mapping).

Example

print(type(123))            # <class 'int'>
print(type(3.14))           # <class 'float'>
print(type('hello'))        # <class 'str'>

L = [1,2,3]
T = (1,2,3)
D = {'name': 'Khushi'}
print(L[0], T[1], D['name'])

Common mistakes

  • Attempting to change a tuple (immutable) results in TypeError.
  • Using unhashable types (like lists) as dict keys or set elements (TypeError).
  • Confusing identity (is) with equality (==).

Exercises

  1. Create a list of numbers, append a value, and print the last element.
  2. Create a tuple and attempt to change an element — observe the error and explain why.
  3. Create a dict of students and their scores, then print names with scores > 80.
