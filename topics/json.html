JSON (JavaScript Object Notation)

JSON is a lightweight text format for structured data. Python's json module converts between Python objects and JSON strings/files.

Basic usage

import json

data = {"name": "Alice", "age": 30, "skills": ["python", "sql"]}

# to JSON string
s = json.dumps(data)

# to file
with open("data.json","w") as f:
    json.dump(data, f, indent=2)

# read back
with open("data.json") as f:
    obj = json.load(f)

Notes

  • JSON supports objects, arrays, strings, numbers, booleans, null.
  • Use indent for pretty output. For streaming large JSON, parse incrementally or use specialized libraries.
