JSON in Python

JSON (JavaScript Object Notation) is used to store and exchange data between systems.

üåê Commonly used in APIs and web applications.

Convert JSON to Python Object

import json

data = '{"name": "Asha", "age": 15}'
obj = json.loads(data)
print(obj["name"])

Convert Python Object to JSON

import json

data = {"city": "Delhi", "pin": 110001}
json_data = json.dumps(data)
print(json_data)
