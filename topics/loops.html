Loops in Python

Loops are used to execute a block of code repeatedly. They help reduce repetition and make programs efficient.

üí° Python mainly uses for loops and while loops.

For Loop

A for loop is used to iterate over a sequence such as a list, string, or range. 

for i in range(1, 6):
    print(i)

While Loop

A while loop runs as long as a condition is true. 

count = 1
while count <= 5:
    print(count)
    count += 1

Loop Control Statements

  • break ‚Äì exits the loop
  • continue ‚Äì skips the current iteration
  • pass ‚Äì does nothing (placeholder)
for i in range(1, 6):
    if i == 3:
        continue
    if i == 5:
        break
    print(i)
