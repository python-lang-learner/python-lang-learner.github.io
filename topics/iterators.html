Iterators & Iterables

An iterable is an object you can loop over (lists, tuples, dicts, sets, strings). An iterator is an object representing a stream of data — it returns the next element with next().

Basics

nums = [1,2,3]
it = iter(nums)         # get iterator
next(it)                # 1
next(it)                # 2

Writing your own iterator (class)

class Count:
    def __init__(self, limit):
        self.i = 0
        self.limit = limit
    def __iter__(self):
        return self
    def __next__(self):
        if self.i >= self.limit:
            raise StopIteration
        val = self.i
        self.i += 1
        return val

for n in Count(3):
    print(n)  # 0,1,2

Generator functions (easy iterators)

Generators are functions using yield — they return an iterator.

def squares(n):
    for i in range(n):
        yield i*i

for s in squares(5):
    print(s)

When to use them

  • Large or infinite streams of data (avoid building huge lists)
  • Lazy evaluation and pipelines
