Arrays in Python (lists & array module)

Python doesn't have a separate "array" type like some languages — the most commonly used sequence is list. For memory-efficient numeric arrays you can use the array module or NumPy (external library).

1. Lists — the Python "array-like"

Lists are dynamic, ordered, and can contain mixed types.

# create lists
fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4]

# add, remove, index
fruits.append("date")
first = fruits[0]          # "apple"
last = fruits[-1]          # "date"
fruits.remove("banana")

Common operations

  • len(list) — length
  • list.append(x), list.insert(i,x)
  • list.pop() — remove last or pop(i)
  • slicing: list[1:4], list[::-1]

2. Numeric arrays (array module)

The built-in array module is for compact numeric arrays of a single type.

from array import array
a = array('i', [1, 2, 3])   # 'i' = signed int
a.append(4)
a[0] = 10

3. When to use NumPy

Use NumPy when you need fast vectorized operations, multi-dimensional arrays, or heavy numeric work (scientific computing).
