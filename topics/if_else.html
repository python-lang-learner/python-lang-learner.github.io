If–Else Statements

if statements execute code based on conditions.

Basic example

x = 10

if x > 5:
    print("Greater")
else:
    print("Smaller")

elif

score = 72

if score >= 90:
    print("A")
elif score >= 75:
    print("B")
elif score >= 50:
    print("C")
else:
    print("Fail")

Nested conditions

x = 20
y = 5

if x > 10:
    if y > 0:
        print("Valid pair")
