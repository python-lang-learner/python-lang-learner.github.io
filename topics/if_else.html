Python Conditional Statements

Decision making using if, elif and else

What is if–else?

In Python, if–else statements are used to make decisions. The program checks a condition and executes different blocks of code depending on whether the condition is True or False.

1. if Statement

The if statement runs code only when the condition is true.

age = 18

if age >= 18:
    print("You are eligible to vote")
Output:

You are eligible to vote

2. if–else Statement

The else block runs when the if condition is false. 

number = 5

if number % 2 == 0:
    print("Even number")
else:
    print("Odd number")
Output:

Odd number

3. if–elif–else Statement

Use elif to check multiple conditions. 

marks = 75

if marks >= 90:
    print("Grade A")
elif marks >= 60:
    print("Grade B")
else:
    print("Grade C")
Output:

Grade B

Important Rules

  • Conditions must end with a colon :
  • Indentation is mandatory in Python
  • elif can be used multiple times
  • else is optional

Real-Life Example

Checking whether a person can enter a movie theater:

age = 12

if age >= 13:
    print("Entry allowed")
else:
    print("Entry not allowed")
