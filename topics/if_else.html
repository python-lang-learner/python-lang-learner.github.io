Conditional statements — if, elif, else

Basic pattern

if condition1:
    # do A
elif condition2:
    # do B
else:
    # default

Examples

x = 10
if x < 0:
    print("negative")
elif x == 0:
    print("zero")
else:
    print("positive")

Boolean expressions

You can combine conditions:

age = 20
if 13 <= age <= 19:
    print("teen")

Use and, or, not for complex logic.

Ternary (one-line) conditional

status = "even" if x % 2 == 0 else "odd"

Exercises

  1. Given a score 0..100, print grade A/B/C/D/F using if/elif/else.
  2. Write a function that returns whether a year is leap (hint: divisible by 4 but not by 100 unless 400).
