For loops

Use for to iterate over sequences (lists, tuples, strings, ranges, etc.). For loops are fundamental for repeating tasks in Python.

Basic syntax

for item in sequence:
    # use item
    print(item)

Example: iterate over a list

fruits = ["apple", "banana", "cherry"]
for f in fruits:
    print("I like", f)

Using range()

range(n) produces numbers from 0 to n-1. Useful when you need an index.

for i in range(5):
    print(i)  # 0 1 2 3 4

Loop with index (enumerate)

names = ["Ana", "Ben", "Cleo"]
for idx, name in enumerate(names, start=1):
    print(idx, name)

Common patterns

  • Loop and build new list: use list comprehensions when possible
  • Break out early: use break
  • Skip an iteration: use continue
total = 0
for n in range(1, 11):
    if n == 7:
        continue
    total += n
print(total)

Exercises

  2. Given a list of grades, compute the average using a for loop.
  3. Use a loop to build a list of squared numbers from 1..10.
