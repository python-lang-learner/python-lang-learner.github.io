Output in Python

Python prints output using the built-in print() function.

Basic printing

print("Hello World")
print(10)
print(3.14)

Multiple values

print("Name:", "Khushi", "Age:", 20)

End and Separator arguments

print("A", "B", "C", sep="-")     # A-B-C
print("Hello", end="!!!")         # Hello!!!

Exercises

  1. Print 3 words separated by commas.
  2. Print 5 numbers separated with | .
