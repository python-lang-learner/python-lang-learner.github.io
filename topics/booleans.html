Boolean Operators

Booleans are True or False. Operators include and, or, not. Many values have truthiness: non-empty sequences are True, zero/empty are False.

Details

Comparisons produce boolean results: ==, !=, <, > etc. Use 'is' to test identity (e.g., is None) not for value equality. Use parentheses to clarify complex boolean expressions.

Examples

x = 5
print(x > 3 and x < 10)
print(not (x == 6))
print('' or 'fallback')  # returns 'fallback'

Truthiness table (common)

  • False values: 0, 0.0, '', [], {}, None
  • Everything else is considered True

Common mistakes

  • Using is for value equality (e.g., a is b vs a == b).
  • Misreading operator precedence — use parentheses.

Exercises
