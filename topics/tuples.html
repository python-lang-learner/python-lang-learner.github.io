Tuples

Tuples are ordered, immutable sequences. They are useful for fixed collections and for using as dict keys (when elements are hashable).

Details

Create tuples with parentheses or by comma separation. Tuples support indexing and unpacking. Because they are immutable, operations that seem to modify create new tuples instead.

Examples

point = (3, 4)
x, y = point
single = (5,)   # single element tuple
print(x, y)

Common mistakes

  • Expecting methods like append (lists) to exist on tuples.
  • Forgetting trailing comma for single-element tuples.

Exercises

  1. Unpack a tuple of three elements into variables and print them.
  2. Convert a list into a tuple and demonstrate immutability.

Next steps

Use tuples for fixed records. Explore lists when mutability is needed.
