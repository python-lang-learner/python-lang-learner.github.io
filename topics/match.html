Match Statement

match is Python’s version of switch-case (introduced in Python 3.10).

Basic example

day = 3

match day:
    case 1:
        print("Monday")
    case 2:
        print("Tuesday")
    case 3:
        print("Wednesday")
    case _:
        print("Invalid")

Multiple patterns

match letter:
    case 'a' | 'e' | 'i' | 'o' | 'u':
        print("Vowel")
    case _:
        print("Consonant")

Structured matching

person = ("Asha", 20)

match person:
    case (name, age):
        print(name, age)

Exercises

  1. Use match to print month name from number.
  2. Match a tuple (x, y) and identify which quadrant point lies in.
