Structural pattern matching — match (Python 3.10+)

Pattern matching lets you destructure values and handle cases clearly. It's similar to switch/case in other languages but more powerful.

Basic example

def http_status(status):
    match status:
        case 200:
            return "OK"
        case 404:
            return "Not found"
        case _:
            return "Other"

Matching sequences & capturing

match point:
    case (0, 0):
        print("origin")
    case (x, 0):
        print("x on axis:", x)
    case (x, y):
        print("coords:", x, y)

Matching dict-like objects

data = {"type":"user", "id": 5}
match data:
    case {"type":"user", "id": uid}:
        print("User id", uid)
    case _:
        print("Unknown")

When to use

  • Dispatching on structured data
  • Replacing long if/elif chains when matching shapes

Exercises

  1. Write a match that handles 2- and 3-element tuples differently.
  2. Use match to parse simple command dictionaries and return actions.
