Virtual environments

Virtual environments isolate project dependencies so different projects don't conflict.

Create & activate

python -m venv venv        # create
# Windows
venv\Scripts\activate
# mac / linux
source venv/bin/activate

Deactivate

deactivate

Using with pip

pip install -r requirements.txt

Alternatives

  • venv — built-in and widely used
  • virtualenv — older alternative
  • pipx — run CLI tools in isolated envs
  • poetry / pipenv — higher-level tools for dependency + packaging management
