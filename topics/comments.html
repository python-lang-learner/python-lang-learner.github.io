Comments

Comments help explain code. They are ignored by Python during execution.

Single-line comments

# This is a comment
print("Hello")       # This is also a comment

Multi-line comments (docstrings)

"""
This is a multi-line comment or documentation string.
Useful for documenting modules, classes and functions.
"""

Exercises

  1. Add comments to explain a print statement.
  2. Write a function and add a docstring to explain it.
