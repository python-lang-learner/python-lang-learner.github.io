Lists, Tuples & Sets — Exercises

Work with sequence operations, comprehensions, uniqueness and immutability.

1. Lists

  1. Given nums = [1,2,3,4,5] create a new list with squares using a list comprehension.
  2. Remove duplicates from [1,2,2,3,3,3] while preserving order.

2. Tuples

  1. Create a tuple representing a point (x, y) and unpack into x and y.
  2. Convert a tuple to a list, modify it, then convert back to a tuple.

3. Sets

  1. Use sets to find common elements (intersection) of two lists.
  2. Given a sentence string, count unique words using sets (case-insensitive).

4. Mixed challenge

  1. Given list of tuples representing (name, score), produce a dict mapping name → highest score (use loops or comprehensions).
  2. Given a large list of numbers, show how sets can speed up membership tests compared to lists (explain with simple timing code example).

1. Lists

# 1. squares using comprehension
nums = [1,2,3,4,5]
squares = [x*x for x in nums]

# 2. remove duplicates while preserving order
arr = [1,2,2,3,3,3]
seen = set()
result = []
for x in arr:
    if x not in seen:
        seen.add(x)
        result.append(x)
print(result)  # [1,2,3]

4. Mixed challenge

# 7. map name -> highest score
data = [("A", 10), ("B", 20), ("A", 15)]
best = {}
for name, score in data:
    best[name] = max(best.get(name, 0), score)
print(best)

# 8. timing membership difference (simple demonstration)
