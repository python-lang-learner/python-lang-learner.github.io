Regular Expressions (regex)

Use the re module for pattern matching in strings.

Quick examples

import re

# search
m = re.search(r"\d{4}-\d{2}-\d{2}", "2025-12-31")
if m:
    print("Found date:", m.group())

# find all
emails = re.findall(r"[a-zA-Z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}", text)

Useful patterns

  • \d digits, \w word char, \s whitespace
  • Quantifiers: +, *, ?, {n,m}
  • Groups and captures: (...), non-capturing (?:...)

Safety & performance

Avoid catastrophic backtracking by using non-greedy tokens or simpler patterns. Always test complex regex on sample inputs.
