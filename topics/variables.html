Variables

Variables associate names with values. Python uses dynamic typing — variables can hold any type and change type at runtime.

Details

Assign with =. Variable names should start with a letter or underscore, cannot start with a digit and are case-sensitive. Use snake_case for variable names per PEP8. Multiple assignment and tuple-unpacking are common idioms. Keep variable names descriptive and avoid single-letter names except for counters.

Example

x = 10
name = 'Khushi'
a, b = 1, 2
x = 'now a string'
print(x, name, a, b)

Common mistakes

  • Using reserved keywords as variable names (SyntaxError).
  • Confusing local vs global scope leading to UnboundLocalError.
  • Reusing variable names accidentally and losing data.

Exercises

  1. Create variables name, age, and city, then print them in one line.
  2. Swap two variables a and b without using a temporary variable.
  3. Write a small snippet that assigns multiple values in one line and prints them.
