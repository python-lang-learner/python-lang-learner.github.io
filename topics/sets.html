Sets

Sets are unordered collections of unique elements. They are efficient for membership tests and set algebra (union, intersection, difference).

Details

Create sets with curly braces or set(). Elements must be hashable. Useful operations: | (union), & (intersection), - (difference), ^ (symmetric difference).

Examples

a = {1,2,3,3}
print(a)           # {1,2,3}
b = {2,3,4}
print(a & b)       # intersection
print(a | b)       # union

Common mistakes

  • Expecting order in sets — iteration order is arbitrary.
  • Using unhashable items like lists as set elements (TypeError).

Exercises

  1. Remove duplicates from a list using a set and preserve order (hint: use helper structure).
  2. Compute intersection and difference of two lists using sets.
