Modules & Packages

Modules are Python files providing functions, classes and variables. Packages are directories with an __init__.py (optional in modern Python) that group related modules.

Importing

import math
from math import sqrt
import mymodule as mm

Creating a module

Make a file utils.py:

# utils.py
def greet(name):
    return f"Hello, {name}"

Then use it:

from utils import greet
print(greet("Anna"))

Packages

mypkg/
  __init__.py
  math_tools.py
  io_tools.py

Installing & virtual environments

Use pip install package-name. Always use a virtual environment: python -m venv venv then venv\Scripts\activate (Windows) or source venv/bin/activate (mac/linux).
