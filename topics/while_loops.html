While loops

A while loop repeats as long as a condition remains true. Use it for indefinite loops where iteration count is unknown.

Basic syntax

while condition:
    # repeated code
    do_something()

Example: countdown

n = 5
while n > 0:
    print(n)
    n -= 1
print("Lift off!")

Guard against infinite loops

Make sure the loop condition will become false eventually, or use a break with a safe timeout.

attempts = 0
while True:
    attempts += 1
    if attempts > 5:
        break

Using while to read input

# keep reading until user types "quit"
while True:
    s = input("Enter command: ")
    if s == "quit":
        break
    print("You typed:", s)

Exercises

  • Write a while-loop to sum numbers entered by user until they type "0".
  • Create a loop that asks for a password up to 3 times, then stops.
