While Loops

while repeats code as long as the condition is True.

Basic example

i = 1
while i <= 5:
    print(i)
    i += 1

Break & continue

i = 1
while i < 10:
    if i == 5:
        break
    print(i)
    i += 1

Infinite loop (avoid!)

while True:
    print("Running")
    break

Exercises

  1. Print all even numbers from 1 to 50.
  2. Count digits in a number using a while loop.
