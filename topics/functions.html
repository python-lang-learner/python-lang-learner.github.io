Functions

Functions group reusable code. Use def to define a function and return to return a value.

Basic example

def greet(name):
    return "Hello, " + name

print(greet("Asha"))  # Hello, Asha

Arguments & defaults

def power(x, n=2):   # n has default value 2
    return x ** n

print(power(3))    # 9
print(power(3, 3)) # 27

Docstrings

Document your functions with a docstring right under the definition.

def add(a, b):
    """Return the sum of a and b."""
    return a + b
