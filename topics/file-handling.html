File handling

Open, read, write and manage files using the built-in file APIs.

Open / read / write

with open("file.txt", "w", encoding="utf-8") as f:
    f.write("Hello\n")

with open("file.txt", "r", encoding="utf-8") as f:
    text = f.read()

Read lines

with open("file.txt") as f:
    for line in f:
        print(line.strip())

Binary files

with open("image.png", "rb") as f:
    data = f.read()

Paths

from pathlib import Path
p = Path("data")
p.exists()
p.joinpath("file.txt")

Notes

  • Always prefer `with` to ensure files close promptly.
  • Use `pathlib` for cross-platform path operations.
