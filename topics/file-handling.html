File Handling in Python

File handling allows Python programs to read data from files and write data to files. It is useful for storing data permanently.

üí° Files are used to save data even after the program ends.

Opening a File

Use the open() function to open a file. 

file = open("data.txt", "r")

Common file modes:

  • "r" ‚Äì Read
  • "w" ‚Äì Write
  • "a" ‚Äì Append
  • "x" ‚Äì Create

Reading a File

file = open("data.txt", "r")
content = file.read()
print(content)
file.close()

Writing to a File

file = open("data.txt", "w")
file.write("Hello Python!")
file.close()
‚ùó Using "w" mode will overwrite existing content.

Appending to a File

file = open("data.txt", "a")
file.write("\nWelcome to Python Learning")
file.close()

Using with Statement

The with statement automatically closes the file. 

with open("data.txt", "r") as file:
    print(file.read())
