Small Projects to Practice

Build these small projects to solidify the concepts you've learned.

```

1. Calculator (CLI)

Build a command-line calculator supporting +, -, *, / and a loop to continue.

2. To-Do List (file-based)

Save tasks to a text file. Add, list, and delete tasks. Practice file I/O.

3. Simple Web Scraper

Use requests and BeautifulSoup to fetch and parse a webpage (read robots.txt first).

4. Temperature Converter (GUI)

Use tkinter to make a small GUI that converts between °C and °F.

5. Mini Project: Quiz App

Create a CLI quiz app that asks multiple-choice questions stored in a JSON file and scores the user.

```
