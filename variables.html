Python Variables & Types

Learn how to store values and use basic types in Python.

```

What is a variable?

A variable stores a value. Use an equals sign to assign:

name = "Alice"
```

age = 25
is_student = True
```

Common types

  • int — integers: 42
  • float — real numbers: 3.14
  • str — text: "hello"
  • bool — True / False

Examples

# combine values
```

first = "Ada"
last = "Lovelace"
print(first + " " + last)

# arithmetic

x = 10
y = 3
print(x + y, x * y, x / y)
```

Short exercise
```
