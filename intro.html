Intro to Python — first steps

Python is a beginner-friendly, powerful programming language used for web development, data science, automation, AI, and more.

1. Install Python

Download Python (3.10+ recommended) from python.org. On macOS use Homebrew, on Linux use your package manager.

2. Your first Python script

Create a file named hello.py:

print("Hello, Python!")

Run it using:

python hello.py
