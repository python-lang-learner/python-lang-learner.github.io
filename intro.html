    Intro to Python — first steps

    This lesson quickly introduces Python and shows how to run a simple program. Follow the steps below — then head to Syntax for examples.

    ```

    1. Install Python

    Install Python (3.10+ recommended). On Windows download from python.org or use Chocolatey / on macOS use Homebrew.

    2. Your first script

    Create a file named hello.py and paste:

    print("Hello, Python!")

    Run it with python hello.py.
    ```
