Setup — Install Python & Tools

Follow these steps to set up Python and a comfortable development environment.

```

1. Install Python

  1. Go to python.org/downloads.
  2. Download the latest Python 3 installer for your OS (Windows / macOS / Linux).
  3. On Windows, run installer and check "Add Python to PATH" then click Install.

Verify in terminal:

python --version
```

# or on some systems:

python3 --version
```

2. Install VS Code

VS Code is a lightweight editor with excellent Python support.

  1. Download from code.visualstudio.com.
  2. Install the Python extension (Microsoft) inside VS Code.

3. Create & run your first script

Open a terminal or VS Code terminal and run:

echo print("Hello, world!") > hello.py
```

python hello.py

You should see Hello, world! printed.

```

Helpful commands & tips

  • Use python -m venv env then env\Scripts\activate (Windows) or source env/bin/activate (macOS/Linux) to create and use virtual environments.
  • Install packages with pip install package-name.
  • Use pip freeze > requirements.txt to save dependencies.
```
