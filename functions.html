Functions — Write reusable code

Functions help you group logic into reusable blocks.

```

Defining a function

def greet(name):
print("Hello", name)
```

greet("Sam")
```

Return values

def add(a, b):
return a + b
```

result = add(3, 4)
```

Parameters & defaults

def power(x, n=2):
return x ** n
```

print(power(5))   # 25
print(power(2, 3))# 8
