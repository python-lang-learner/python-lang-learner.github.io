Web Scraper — Headlines

Difficulty: Intermediate • Dependencies: requests, beautifulsoup4

Scrapes headlines from a news site. Selector may need adjustment per site.

Download code (.py)

Code (preview)

# scraper.py
# pip install requests beautifulsoup4
import requests
from bs4 import BeautifulSoup

URL = "https://www.bbc.com/news"

def fetch_headlines(url=URL):
    r = requests.get(url, timeout=10)
    r.raise_for_status()
    doc = BeautifulSoup(r.text, "html.parser")
    headlines = []
    for h in doc.select("h3"):
        text = h.get_text(strip=True)
        if text:
            headlines.append(text)
    return headlines[:10]

if __name__ == "__main__":
    for i, h in enumerate(fetch_headlines(), 1):
        print(i, h)

How to run

  1. Install: pip install requests beautifulsoup4
  2. Run: python scraper.py
  3. Note: obey site terms and robots.txt, and adapt selectors as needed.
