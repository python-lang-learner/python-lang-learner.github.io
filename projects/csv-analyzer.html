CSV Data Analyzer

Difficulty: Intermediate • Dependencies: pandas

Load a CSV and print summary statistics and top rows for quick exploration.

Download code (.py)

Code (preview)

# analyzer.py
# pip install pandas
import pandas as pd
from pathlib import Path

FILE = Path("data.csv")  # replace with your CSV

def summarize(path=FILE):
    df = pd.read_csv(path)
    print("Shape:", df.shape)
    print("\nColumns:\n", df.columns.tolist())
    print("\nHead:\n", df.head().to_string(index=False))
    print("\nDescribe:\n", df.describe(include='all').to_string())

if __name__ == "__main__":
    if not FILE.exists():
        print("Place a data.csv file in the current folder.")
    else:
        summarize()

How to run

  1. Install: pip install pandas
  2. Add a `data.csv` file next to `analyzer.py` and run: python analyzer.py
