Simple Flask Notes App

Difficulty: Intermediate • Dependencies: Flask

Tiny web app that stores notes in memory for demo (not persistent). Great to learn routes and templates.

Download code (.py)

Code (preview)

# app.py
# pip install flask
from flask import Flask, request, redirect, render_template_string

app = Flask(__name__)
notes = []

TEMPLATE = '''
Notes

  
Notes

  

    
    
  

  
    
    {% for n in notes %}
      
  • {{ n }}
    • 
    {% endfor %}
  


'''

@app.route('/')
def index():
    return render_template_string(TEMPLATE, notes=notes)

@app.route('/add', methods=['POST'])
def add():
    text = request.form.get('text','').strip()
    if text:
        notes.append(text)
    return redirect('/')

if __name__ == '__main__':
    app.run(debug=True)

How to run

  1. Install: pip install flask
  2. Run: python app.py and open http://127.0.0.1:5000/
  3. Note: This demo app stores notes in memory — restart loses data.
